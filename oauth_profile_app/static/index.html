🛠️ Developer Integration Guide

To test the login flow, you must register OAuth applications and configure credentials in a local .env file:

  • GitHub: Go to GitHub ➡️ Developer Settings ➡️ OAuth Apps. Set callback URL to http://localhost:8000/api/auth/github/callback.
  • Google: Go to Google Cloud Console ➡️ Credentials ➡️ Create OAuth Client ID. Set redirect URI to http://localhost:8000/api/auth/google/callback.
